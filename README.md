# TokenSite

Solana-powered crypto landing page generator.

## Requirements
- VPS with Docker + Docker Compose
- Traefik reverse proxy running externally
- Cloudflare DNS with wildcard record: `*.tokensite.fun → VPS IP`

- Service Availability & Deletion

Our platform is designed for short-term and fast-moving projects. If a service is not renewed before its expiration date, it is automatically deleted.

This includes all associated data, content, and settings, which are permanently removed and cannot be recovered. This system is intentional and ensures the platform remains clean and efficient while allowing names and slugs to be reused for future projects.
